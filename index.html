To install the extension:

  • Click the above download link. It will automatically download a file named slimfo.crx into your default download folder.
  • Navigate to chrome://extensions in Google Chrome
  • Open your default downloads folder and find the slimfo.crx file.
  • Drag and drop the file into your browser window.
